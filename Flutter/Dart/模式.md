Dart 3 引入的模式 (Patterns)是一项重大语言特性，用于解构数据结构、匹配值，让代码更简洁、安全。

模式是一组语法规则，用于**检查**值的结构（匹配）、**提取**值的部分内容（解构），或两者兼顾。

```dart
// 不用模式
var list = [1, 2];
var a = list[0];
var b = list[1];

// 使用列表模式
var [a, b] = [1, 2];  // a=1, b=2
```

### 模式的使用场景

| 场景 | 示例 |
|------|------|
| 变量声明/赋值 | `var (x, y) = (1, 2)` |
| `for` 循环 | `for (var [key, value] in map.entries)` |
| `if` 条件判断 | `if (obj case int i)` |
| `switch` 语句/表达式 | `switch (obj) { case int i: ... }` |
| 解构 | `final (first, second) = record` |

## 常见模式类型

### 1. 匹配模式 (Match)

检查值是否符合某种类型或条件。

```dart
// 类型匹配
void printType(Object obj) {
  switch (obj) {
    case int i: print('整数: $i');
    case String s: print('字符串: $s');
    case List l: print('列表长度: ${l.length}');
    default: print('其他');
  }
}

// 守卫子句 (带条件)
switch (obj) {
  case int i when i > 0: print('正整数');
  case int i when i < 0: print('负整数');
  case 0: print('零');
}
```

### 2. 解构模式 (Destructuring)

提取复合类型内部的值。

```dart
// 记录解构
var (name, age) = ('Alice', 25);
print('$name 今年 $age 岁');

// 带命名字段
var (name: n, age: a) = (name: 'Bob', age: 30);
print('$n, $a');

// 列表解构
var [first, second] = [10, 20];
var [head, ...tail] = [1, 2, 3, 4];  // head=1, tail=[2,3,4]
var [a, b, ...rest] = [5, 6, 7, 8, 9];  // rest=[7,8,9]
// _忽略不需要的值
var [_, two, _] = [100, 200, 300];  // two=200

// Map 解构
var {'name': n, 'age': a} = {'name': 'Charlie', 'age': 28, 'city': 'NYC'};

// for-in 循环
Map<String, int> hist = {'a': 23, 'b': 100};
for (var MapEntry(key: k, value: count) in hist.entries) {
  print('$k occurred $count times');
}
```

### 3. 对象模式

提取对象的属性。

```dart
class Point {
  final double x, y;
  const Point(this.x, this.y);
}

void main() {
  var p = Point(3, 4);
  
  // 对象模式解构
  var Point(x: x1, y: y1) = p;
  print('x=$x1, y=$y1');  // x=3, y=4
  
  // 可以只提取部分属性
  var Point(x: x2) = p;
  print('x=$x2');  // x=3
  
  // switch 中使用
  switch (p) {
    case Point(x: 0, y: 0): print('原点');
    case Point(x: 0, y: var y): print('Y轴上: $y');
    case Point(x: var x, y: 0): print('X轴上: $x');
    case Point(x: var x, y: var y): print('点($x, $y)');
  }
}
```

### 4. 可变模式 (Var)

提取值到新变量。

```dart
// var 模式可以提取任意值
switch (obj) {
  case var v: print('值是: $v');  // 匹配一切
}

// 与类型结合
switch (obj) {
  case int var i: print('整数: $i');
  case [var a, var b]: print('两个元素的列表: $a, $b');
}
```

### 5. 逻辑模式

组合多个模式。

```dart
// && (且)
switch (obj) {
  case int i && ( > 0 && < 10):
    print('1-9之间的整数');
}

// || (或)
var [a, b] = [1, 2];
if (a == 1 || b == 2) { ... }

// 模式中的或
switch (obj) {
  case 0 || 1 || 2:
    print('0,1,2 之一');
}
```

### 嵌套模式

模式可以任意嵌套，处理复杂数据结构：

```dart
var data = [
  {'name': 'Alice', 'scores': [95, 87, 92]},
  {'name': 'Bob', 'scores': [78, 85, 88]},
];

for (var [{'name': name, 'scores': [first, ...rest]}] in [data]) {
  print('$name 第一门成绩: $first, 剩余: $rest');
}
// 输出: Alice 第一门成绩: 95, 剩余: [87, 92]
//       Bob 第一门成绩: 78, 剩余: [85, 88]
```

---

## 总结

| 模式类型 | 语法示例 | 主要用途 |
|----------|----------|----------|
| 匹配 | `case int i` | 类型检查并提取 |
| 解构 | `var [a,b] = list` | 拆分复合数据 |
| 通配符 | `_` | 忽略不需要的值 |
| 常量 | `case 42` | 精确值匹配 |
| 变量 | `case var v` | 捕获任意值 |
| 逻辑 | `&&`、`\|\|` | 组合条件 |
| 对象 | `Point(x: x, y: y)` | 提取对象属性 |

模式让 Dart 代码从**命令式取值**转向**声明式匹配**，特别适合处理 JSON、状态管理、抽象语法树等场景。
