## 条件分支
### `if-else`
注意 Dart 的布尔值必须是 `bool` 类型。

```dart
int score = 85;

if (score >= 90) {
  print('优秀');
} else if (score >= 60) {
  print('及格');
} else {
  print('不及格');
}
```

### `switch-case`

```dart
// 传统用法
String grade = 'A';
switch (grade) {
  case 'A':
    print('优秀');
    break;  // 必须显式 break
  case 'B':
    print('良好');
    break;
  default:
    print('未知');
}

// Dart 3+ 增强：支持模式匹配
dynamic value = 42;
switch (value) {
  case 0:
    print('零');
  case int i when i > 0:
    print('正整数: $i');
  case int i:
    print('负整数: $i');
  case String s:
    print('字符串长度: ${s.length}');
  case [var a, var b]:
    print('两元素列表: $a, $b');
  case {'name': String name}:
    print('包含 name 字段的 Map: $name');
  default:
    print('其他');
}

// switch 表达式 (Dart 3.0+)
String result = switch (value) {
  0 => '零',
  int i when i > 0 => '正数',
  int i => '负数',
  _ => '其他',
};
```

## 循环

### `for` 循环
经典三段式循环。

```dart
// 标准 for
for (int i = 0; i < 5; i++) {
  print('第 $i 次');
}
```

### `for-in` 循环
遍历可迭代对象（List、Set、Map 等）。

```dart
var fruits = ['苹果', '香蕉', '橙子'];

for (var fruit in fruits) {
  print(fruit);
}

// 遍历 Map
var map = {'a': 1, 'b': 2, 'c': 3};
for (var entry in map.entries) {
  print('${entry.key}: ${entry.value}');
}

// 使用 forEach（函数式风格）
fruits.forEach((fruit) => print(fruit));
```

### `while` 和 `do-while`

```dart
int count = 0;
while (count < 3) {
  print('while: $count');
  count++;
}

int n = 5;
do {
  print('do-while: $n');
  n--;
} while (n > 0);
```

## 循环控制

### `break` 和 `continue`

```dart
for (int i = 0; i < 10; i++) {
  if (i == 5) break;
  print(i);  // 0,1,2,3,4
}

for (int i = 0; i < 5; i++) {
  if (i == 2) continue;
  print(i);  // 0,1,3,4
}

// 标签（跳出多层循环）
outerLoop:
for (int i = 0; i < 3; i++) {
  for (int j = 0; j < 3; j++) {
    if (i == 1 && j == 1) break outerLoop;
    print('($i, $j)');
  }
}
// 输出: (0,0) (0,1) (0,2) (1,0)
```




## 6. 空安全相关控制流

### `if-case` 模式匹配

```dart
// 检查并解构
var maybeList = [1, 2, 3];

if (maybeList case [int first, ...]) {
  print('第一个元素是: $first');
}

// 结合守卫
if (maybeList case [var first, var second] when first > second) {
  print('第一个大于第二个');
}
```

### `switch` 作为表达式

```dart
// 直接返回值
String describe(dynamic value) {
  return switch (value) {
    null => '空值',
    int i when i > 0 => '正整数',
    int i when i < 0 => '负整数',
    0 => '零',
    String s => '字符串: $s',
    _ => '其他类型',
  };
}
```
