

Dart 是真正的面向对象语言，函数也是对象，类型为 `Function`。这意味着函数可以赋值给变量、作为参数传递。

### 基本语法

```dart
int add(int a, int b) {
  return a + b;
}

// 省略返回类型（不推荐）
add(int a, int b) {
  return a + b;  // 返回类型被推断为 int
}

// 如果未指定返回值，则语句 `return null;` 会隐式添加到函数体中。
foo() {}

assert(foo() == null);
```

### 简写语法（表达式函数体）
`=> expr` 语法是 `{ return expr; }` 的简写。`=>` 符号有时被称为 arrow 语法。

```dart
// 单行表达式可以使用 =>
int add(int a, int b) => a + b;

// 等价于
int add(int a, int b) {
  return a + b;
}

String greet(String name) => 'Hello, $name!';

// 集合操作中的常见用法
var list = [1, 2, 3, 4];
list.forEach((item) => print(item));
var squared = list.map((x) => x * x).toList();
```

## 参数

### 必需位置参数

```dart
// 位置参数（必需）
void register(String username, String password) {
  print('注册: $username');
}
```

### 可选参数

Dart 支持两种可选参数：**位置可选** 和 **命名可选**，且可以设置默认值。

#### 可选位置参数（使用 `[]`）

```dart
// 可选参数必须放在必需参数后面
String greet(String name, [String? title, String? suffix]) {
  if (title != null) {
    return 'Hello, $title $name$suffix';
  }
  return 'Hello, $name$suffix';
}

print(greet('Alice'));                    // Hello, Alice
print(greet('Alice', 'Ms.'));             // Hello, Ms. Alice
print(greet('Alice', 'Ms.', '!'));        // Hello, Ms. Alice!

// 带默认值
String sayHi(String name, [String prefix = 'Mr.']) {
  return 'Hi, $prefix $name';
}
```

#### 可选命名参数（使用 `{}`）
如果没有标记为 `required`，类型必须是可为空的，因为它们的默认值将是 `null`

```dart
void createUser({
  required String username,  // required 表示必需
  String? email,             // 可选
  int age = 18,              // 带默认值
  String? phone,
}) {
  print('用户名: $username');
  if (email != null) print('邮箱: $email');
  print('年龄: $age');
}

// 调用时使用参数名
createUser(
  username: 'alice',
  email: 'alice@example.com',
  age: 25,
);

// required 参数必须提供
// createUser();  // ❌ 缺少 username
```

### 混合使用

```dart
// 必需位置参数 + 可选命名参数
void configure(String url, {int timeout = 30, bool retry = false}) {
  print('连接 $url，超时 ${timeout}s，重试: $retry');
}

configure('https://api.example.com');
configure('https://api.example.com', timeout: 60, retry: true);
```
## 函数作为一等公民

### 赋值给变量

```dart
// 函数字面量
var add = (int a, int b) => a + b;
print(add(3, 5));  // 8

// 显式类型声明
int Function(int, int) multiply = (a, b) => a * b;
print(multiply(4, 5));  // 20

// 函数引用
int subtract(int a, int b) => a - b;
var myFunc = subtract;
print(myFunc(10, 3));  // 7
```

### 作为参数传递

```dart
// 接收函数作为参数
void forEach(List<int> list, void Function(int) action) {
  for (var item in list) {
    action(item);
  }
}

// 使用
var numbers = [1, 2, 3, 4];
forEach(numbers, (n) => print(n * 2));  // 2,4,6,8

// 内置高阶函数示例
var evens = numbers.where((n) => n.isEven).toList();  // [2,4]
var doubled = numbers.map((n) => n * 2).toList();     // [2,4,6,8]
var sum = numbers.reduce((a, b) => a + b);            // 10
```

### 返回函数（闭包）

```dart
// 函数返回函数
Function makeMultiplier(int factor) {
  return (int x) => x * factor;
}

var double = makeMultiplier(2);
var triple = makeMultiplier(3);

print(double(5));  // 10
print(triple(5));  // 15

// 闭包示例：记住状态
Function counter() {
  int count = 0;
  return () => ++count;
}

var myCounter = counter();
print(myCounter());  // 1
print(myCounter());  // 2
print(myCounter());  // 3
```

## 匿名函数（Lambda）

```
([[Type] param1[, ...]]) {
  codeBlock;
}
```

```dart
// 多参数，多行语句
var calculate = (int a, int b) {
  var sum = a + b;
  var product = a * b;
  return '和: $sum, 积: $product';
};
print(calculate(3, 4));  // 和: 7, 积: 12

// 无参数
var sayHello = () => print('Hello');
sayHello();

// 单参数
var square = (int x) => x * x;
print(square(5));  // 25
```

## Tear-offs

当你引用函数、方法或命名构造函数时，不带括号，Dart 会创建一个 _tear-off_。这是一个闭包，它接受与该函数相同的参数，并在你调用它时调用底层函数。

```dart
var charCodes = [68, 97, 114, 116];
var buffer = StringBuffer();

// --- good ---
// Tear-off：复用同一个函数对象
// Function tear-off
charCodes.forEach(print);

// Method tear-off
charCodes.forEach(buffer.write);

// ---bad---
// 闭包：每次创建新函数对象
// Function lambda
charCodes.forEach((code) {
  print(code);
});

// Method lambda
charCodes.forEach((code) {
  buffer.write(code);
});
```
## 生成器函数

### 同步生成器（`sync*`）

```dart
// 生成可迭代对象
Iterable<int> naturalsTo(int n) sync* {
  for (int i = 1; i <= n; i++) {
    yield i;  // 产生一个值
  }
}

// 递归生成器
Iterable<int> fibonacci(int n) sync* {
  if (n <= 0) return;
  yield n;
  yield* fibonacci(n - 1);  // yield* 产生另一个序列
}

var numbers = naturalsTo(5);
print(numbers.toList());  // [1, 2, 3, 4, 5]

for (var n in naturalsTo(3)) {
  print(n);  // 1,2,3
}
```

### 异步生成器（`async*`）

```dart
// 生成 Stream
Stream<int> countStream(int max) async* {
  for (int i = 1; i <= max; i++) {
    await Future.delayed(Duration(seconds: 1));
    yield i;  // 异步产生值
  }
}

// 使用
void main() async {
  await for (var value in countStream(5)) {
    print(value);  // 每秒输出 1,2,3,4,5
  }
}
```
