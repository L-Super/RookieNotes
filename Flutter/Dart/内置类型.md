在 Dart 中，所有东西都是对象（继承自 `Object`），因此内置类型的变量本质上也是对象，可以包含方法和属性。

- 数字 (`int`, `double`)
- 字符串 (`String`)
- 布尔值 (`bool`)
- 记录 (`(value1, value2)`)
- 函数 (`Function`)
- 列表 (`List`)
- 集合 (`Set`)
- 映射 (`Map`)
- Runes (`Runes`；常被 `characters` API 替换)
- Symbol (`Symbol`)
- 值 `null` (`Null`)

还有一些其他类型在 Dart 语言中也扮演着特殊角色

- `Object`：除 `Null` 外所有 Dart 类的父类。
- `Enum`：所有枚举的父类。
- `Future` 和 `Stream`：用于[异步编程](https://dart.ac.cn/language/async)。
- `Iterable`：用于[for-in 循环](https://dart.ac.cn/libraries/dart-core#iteration)和同步[生成器函数](https://dart.ac.cn/language/functions#generators)。
- `Never`：表示表达式永远无法成功完成求值。最常用于总是抛出异常的函数。
- `dynamic`：表示你希望禁用静态检查。通常你应该改用 `Object` 或 `Object?`。
- `void`：表示一个值永远不会被使用。常用于作为返回类型。

### 数字类型 (`num`, `int`, `double`)
- `num` 是抽象父类。包含基本运算符，例如 +、-、/ 和 *，以及 `abs()`、`ceil()` 和 `floor()` 等方法。
- `int`：不超 64 位的整数，在原生平台，值的范围是 $-2^{63}$ 到 $2^{63}-1$。在 Web 上，表示为 JavaScript 数字，范围是 $-2^{53}$ 到 $2^{53}-1$。即编译为 JS 时是 53 位浮点数。
- `double`：64 位双精度浮点数。

```dart
int age = 25;
double pi = 3.14159;
num anyNumber = 100; // 可以是 int 或 double

// 使用一个或多个下划线 (_) 作为数字分隔符，使长数字字面量更易读
var n1 = 1_000_000;

// 常用属性
print(pi.round());   // 3
print(age.isEven);   // false
```

字符串、数字互转：

```dart
// String -> int
var one = int.parse('1');
assert(one == 1);

// String -> double
var onePointOne = double.parse('1.1');
assert(onePointOne == 1.1);

// int -> String
String oneAsString = 1.toString();
assert(oneAsString == '1');

// double -> String
String piAsString = 3.14159.toStringAsFixed(2);
assert(piAsString == '3.14');
```

### 字符串 (`String`)

Dart 字符串是 UTF-16 编码。支持单引号、双引号、多行字符串和字符串插值。

可以使用 `${expression}` 将表达式的值放入字符串中。如果表达式是一个标识符，你可以省略 `{}`。

```dart
String single = 'Hello';
String double = "World";
String multiLine = '''第一行
第二行''';

// 插值
String name = "Dart";
String text = "学习 $name"; // 学习 Dart
String expr = "2+3=${2+3}"; // 2+3=5

// 原始字符串 (忽略转义)
String raw = r'换行符\n在这里无效';
```

### 布尔值 (`bool`)
只有 `true` 和 `false`。**Dart 是类型安全的**，`if (非bool值)` 会报错。

```dart
bool isPublic = true;
if (isPublic) {
  print('公开');
}
// ❌ 错误：if (1) 编译不通过
```

### 列表 (`List`)
Dart 会推断类型，也支持泛型指定。

```dart
// 推断类型
var list = [1, 2, 3];

// 指定类型
List<String> names = ['Alice', 'Bob'];

// 不可变列表 (编译时常量)
var constantList = const [1, 2, 3];

// 扩展操作符
var newList = [0, ...list]; // [0,1,2,3]
var nullList = null;
var safeList = [...?nullList]; // []
```

### 集合 (`Set`)
无序、唯一的元素集合。大括号前加 `<类型>` 可避免与 Map 冲突。

```dart
var halogens = {'氟', '氯', '溴'}; // 自动推断 Set<String>
Set<int> numbers = {1, 2, 3};

// 添加元素
halogens.add('碘');

// 去重特性
var dupList = [1, 1, 2];
var uniqueSet = {...dupList}; // {1, 2}
```

### 映射 (`Map`)
键值对集合。键和值都可以是任意类型。

```dart
var gifts = {
  'first': '娃娃',
  'second': '汽车',
};

// 传统构造
Map<String, String> students = {};

// 赋值与访问
gifts['third'] = '飞机';
print(gifts['first']); // 娃娃

// 如果键不存在返回 null
print(gifts['fourth']); // null
```

### 符文 (`Runes`)
用于表示 UTF-32 编码的字符串字符。由于 Dart 字符串是 UTF-16，需要 Runes 处理 Emoji 等特殊字符。

```dart
String heart = '♥️';
String emoji = '😊';

// 通过 runes 属性获取 32 位字符
print(emoji.runes); // (128522)

// 从 Runes 构造字符串
Runes input = Runes('\u{1f600}');
String smiley = String.fromCharCodes(input); // 😀
```

### `Symbol`
用于表示 Dart 中的运算符或标识符，通常在反射或 minification 场景使用。日常开发很少直接用。

```dart
var symbol = #radar; // 符号字面量
print(symbol); // Symbol("radar")

// 使用场景：通过反射获取类信息 (需 dart:mirrors)
```

## 特别补充：记录 (`Record`)
Dart 3 引入的新特性，并非对象（不继承 Object），用于打包多个值。

```dart
// 位置字段
var record = ('first', 2, true);
print(record.$1); // 'first'

// 命名字段
var named = {name: 'Alice', age: 25};
print(named.name); // 'Alice'

// 类型注解
(int, String) swap((String, int) pair) {
  return (pair.$2, pair.$1);
}
```

## 类型判断与转换
```dart
// is 关键字检查类型
if (age is int) { ... }

// as 进行类型转换
(num something) {
  int value = something as int;
}

// 显式转换 (字符串转数字)
String numStr = "123";
int parsed = int.parse(numStr);
double doubleParsed = double.parse("3.14");
```
