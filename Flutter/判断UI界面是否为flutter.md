Android 开发者选项（最直观）

`设置 → 开发者选项 → 显示布局边界`
- 原生 Android App：能看到 View 层级边界线
- Flutter App：只能看到一个整体的 Surface/Canvas，内部 UI 元素没有原生 View 边界

效果对比：

```
原生 App:          Flutter App:
┌──────────┐      ┌──────────┐
│┌────────┐│      │          │
││TextView││      │  (空白)  │
│└────────┘│      │  无边界  │
│┌────────┐│      │          │
││Button  ││      │          │
│└────────┘│      └──────────┘
└──────────┘      
有明显层级边界      没有内部边界线 ← Flutter特征
```
